Homework Completion

Link to file

This was less of a project and more of a simple script I created in an hour in order to automate clicking 'Complete Homework' on Moodle. It is a webcrawler that logs into moodle and checks for any incomplete homework. A prompt follows to ask if you want to complete all the homework. I considered changing this from a console application to one with a GUI but felt like it was unnecessary. This script does not do much apart from, well, completing the homework so a user interface will not make it easier to use.

Requires geckodriver, BeautifulSoup, numpy (decided to use numpy just because it simplified the problem of using 2D arrays and I already have it installed) and selenium to be installed.
